== Description ==

### Winsome Gallery by http://abdullahmasum.elance.com/

This plugin will enable image and youtube-video gallery on your wordpress site .

Plugin Features

* Fully customizable.
* Unlimited image gallery.
* Unlimited youtube-video gallery.
* Mobile touch option. 
* Very Lightweight.
* Works on every modern browser.
* Gallery controll



== Installation ==
<pre>
* Install it as a regular WordPress plugin
* After installing this plugin you can use these shortcodes
   1.[wws_igallery]
   2.[wws_igallery category="your-category-name"]

   3.[wws_ygallery]
   4.[wws_ygallery category="your-category-name"]

</pre>
